# AWESOMEWM-Dotfiles
-----

## Installation:
1. Git clone the repo.
2. copy all the stuff from the cloned repo to 

   ` ~/.config/awesome`

    in any case, you don't have this directory. Make it using,
`sudo mkdir ~/.config/awesome`
